Troubleshooting

Start here for common questions around scanning, imports, and follow-up generation.

The scanner is not available

Live scanning requires a supported physical iPhone. In the simulator, use manual add or the signature import flow to keep testing.

Follow-up drafts are unavailable

If draft generation fails during development, make sure the local backend is running and the app can reach the configured API base URL.

Imported contact looks incomplete

That usually means the original source had limited data. You can edit the contact in Met and add context, tags, and notes manually.

Signature import missed a field

Try pasting the signature as text instead of using an image. You can also correct the fields directly on the review screen before saving.
